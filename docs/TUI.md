# TUI 界面

Agent Swarm 使用 Ink 提供现代化的终端用户界面。

## 界面设计

### 状态栏
```
Agent Swarm | manager                    Session: ink-1234 | ● Ready
```

### 消息格式
```
● 你: 你的消息

● manager: AI 响应
```

### 功能
- 🎨 美观的界面设计
- 📝 Markdown 渲染
- ⚡ 流式输出
- 🎯 命令自动完成

## 技术栈

- Ink - React for CLI
- React 18
- TypeScript

## 快捷键

- `Ctrl+C` 两次 - 退出
- `Tab` - 自动完成
- `/help` - 查看帮助
