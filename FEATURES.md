# Agent Swarm 功能清单

**版本**: v0.1.0
**更新日期**: 2026-03-08
**项目状态**: 🟢 活跃开发中

---

## ✅ 已实现功能 (v0.1.0)

### 核心能力

- [x] **多 Agent 管理**
  - 懒加载：收到消息时才启动 Agent
  - 空闲清理：30 分钟无活动自动卸载
  - 生命周期管理：spawn、process、destroy

- [x] **统一消息路由**
  - 单播：点对点消息（`to: "agent-a"`）
  - 广播：一对多消息（`to: ["agent-a", "agent-b"]`）
  - 工作流编排：oncomplete、onerror

- [x] **会话持久化**
  - JSONL 格式存储
  - 上下文恢复（最近 20 条消息）
  - 自动清理过期会话

### 渠道接入

- [x] **CLI 命令行**：交互式命令行界面
- [x] **钉钉**：企业内部应用集成
- [x] **飞书**：企业自建应用集成

### 可靠性特性

- [x] **ACK 追踪**：确认消息投递
- [x] **消息重试**：自动重试失败消息
- [x] **死信队列**：钉钉消息持久化

### 配置管理

- [x] **多层 API Key 配置**
  - 环境变量（优先级 1）
  - 全局配置（优先级 2）
  - Agent 专用（优先级 3）

- [x] **模型配置**
  - 支持多 Provider（Anthropic、OpenAI、GLM）
  - 自定义 Base URL
